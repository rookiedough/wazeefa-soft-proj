# Wazeefa

Wazeefa is a demo recruitment web app that shows how an AI-assisted hiring workflow could work.

## Features

- CV upload and processing
- Candidate scoring and ranking
- Candidate search and filtering
- Dashboard overview
- Candidate profile details
- Interview scheduling and stage tracking
- Notes and rejection actions
- Basic user management
- Insights page with demo metrics

## Note

This is a demo only. Some data and insights are simulated or estimated and should not be used for real hiring decisions.

## Tech Stack

- HTML
- CSS
- JavaScript
- LocalStorage
- JSON sample data

## Usage

Open the app, log in with a demo user, upload or review candidates, and explore the dashboard, insights, candidates, CV upload, and user management pages.
